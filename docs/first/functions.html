# 系统函数

系统定义的函数,可调用

函数名 参数 返回值 说明
request name[,value] string|array 获取设置 get,post,session,server
cache name[,value] string|array 获取设置缓存
uploadfile 参考上传文件 无 上传文件
guid 无 无 生成 Guid
escapeString string|array string 过滤字符串中的 sql
safe_replace string string 将 url 或 html 中的字符替换
randChar num[,format] string 随机生成指定位数字符串
vercode 参见验证码 image 生成验证码
checkvcode string bool 验证验证码是否正确
imageWaterMark 参见图像处理 无 图片水印
sendmail 参见发送邮件 无 发送邮件
url string string 根据参数生成 url
config string string|array 获取配置信息
db 无 无 返回实例化的数据库对象
model string heephp\model 表名，返回模型
lang string string 传入 tag 输出语言包对应值
aop tag[,parms] 无 aop 标签定义
import file[,var] 无 视图中包含文件，可选传入变量列表
transfer_time int string 根据传入的值返回‘刚刚’‘小时之前’..
sstr str,max string 从开始截取指定数量字符串
htmlencode string string 编码的 html
htmldecode string string 解码 html
