    # 代码规范

    heephp代码和文件名使用小写字母开头的小写字母_下划线组合。
    特殊情况下使用小写字母开头，驼峰写法。

    # 文件命名

    文件的命名采用：小写字母+下划线写法

    # 控制器文件名

    控制器文件名采用：小写字母+下划线.php
    控制器类的命名采用：和文件名同名的类名

    # 模型文件名

    模型文件名采用：小写字母+下划线.php
    模型类的命名采用：和文件名同名的类名，通常类名为数据库表名

    # 视图文件夹与文件

    视图文件位于应用的view文件夹
    每一个控制器应存在同一个文件夹，文件夹应该与控制器同名。
    视图文件名采用：小写字母+下划线.php
    通常应与控制器方法同名\

    # 命名空间

    # 控制器

    控制器的命名空间：heephp\{app}\controller;

    所有控制器需继承：heephp\controller;

    # 模型

    模型的命名空间：heephp\{app}\model;

    所有模型需继承：heephp\model;

    # 类与变量

    类与变量使用小写字母开头的小写字母_下划线组合。

    # 常量

    常量通常使用大写字母定义

    # 函数

    函数名使用小写字母开头的小写字母_下划线组合。

    # 配置

    配置项使用小写字母开头的小写字母_下划线组合。
