# 快速上手

通过第一个 HeePHP 程序，您将能了解到 HeePHP 是如何运行的

下面就从一个Hello World！的程序开始吧！

# 第一步：下载程序

您可通过 GitHub 码云下载相关代码：GitHub 码云

下载后解压到指定目录

# 第二步：环境部署

下载 PHPStudy 、XAMPP、WampServer、宝塔等集成环境，可快速搭建运行环境。

PHP 版本选择 7.3，并设置网站根目录为 HeePHP 根目录

# 第三步：运行

如果您看到如下界面，说明已经成功运行了 HeePHP：

# 第四步：修改控制器文件

文件位置\app\index\index.php

控制器内包含了一个 index 方法，即是您看到的界面

use heephp\controller;

}

下面将这个控制器略微调整，增加如下代码

    function hello(){
        return "Hello Word!";

修改后的文件内容如下：

# 第五步：浏览并运行

在修改完之上的文件后，就可以在浏览器内运行了。

URL 路径为 http://{localhost}/index/hello

如果页面出现:

Hello Word!

说明就运行成功了
