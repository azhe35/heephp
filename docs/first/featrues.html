# 特性

HeePHP采用较多现代软件设计思想，使用起来能更加快速上手，运行起来能更快速。

# MVC架构

MVC是先进的具有松散耦合的软件设计架构，对于开发人员更加友好。

# 数据模型

自动加载数据库、自动读取字段、主键，软删除、自动设置读取、写入数据，简单的配置即可使用关联模型。

# 性能优越的视图层

采用原生PHP为视图层，通过import函数即可实现Layout，免于解析的模板，能提升性能超过30%。

# 控制器

控制器采用现代先进的架构，使页面显示的调用更加灵活。fetch success error json 函数使得控制更加容易。

# 自动验证

通过设置自动验证字段以及验证规则、验证错误的提示，在进行增改数据时会自动验证输入的数据。通过配置验证错误页，可自定义验证错误的提示页。

# 自动数据

通过设置模型层的 set get 函数，在写入、读取数据时自动获取或设置数据库的数据，免于在控制器层处理。

# 自动关联模型

通过设置模型层的关联模型以及字段可以自动关联相关模型，对一、对多、多对多关联免于冗余代码的堆积。

# 软删除

通过模型层开启软删除功能，避免数据意外被删除的风险，可通过回收功能对数据分层级管理。

# 缓存

通过配置缓存，可进行文件、Redis、Memcache的缓存，在设置缓存超时时间后，灵活使用缓存功能，以提高系统性能。

# 面向切面

通过定义切面位置，以及调用切面函数，引用传参、数据返回，方便的拓展系统功能。

# Web应用套装

上传、分页、验证码、图片处理、多语言，一应俱全，得心应手的开发从此变成现实，函数式的调用方便快捷。

# 安全

输入数据的过滤、数据库侵入过滤、脚本注入过滤、上传的文件验证，使得系统更加安全可靠。
