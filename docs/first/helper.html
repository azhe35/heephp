# 助手函数

系统定义的函数,可调用

函数名 参数 返回值 说明
request name[,value] string|array 获取设置 get,post,session,server
cache name[,value] string|array 获取设置缓存
url string[,parms,suffix] string 根据路径生成 url
config string string|array 获取配置信息
db 无 无 返回实例化的数据库对象
table string heephp\orm\orm 返回实例化的 ORM 对象 ,参数为表名
model string heephp\model 返回模型，参数模型名
lang string string 传入 tag 输出语言包对应值
aop tag[,parms] 无 aop 标签定义
import file[,var] 无 视图中包含文件，可选传入变量列表
widget path,parms 无 视图中调用任意控制器方法，作为 widget 小部件
json array string 将数组转为 json 字符串
