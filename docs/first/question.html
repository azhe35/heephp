# 常见问题

# HeePHP容易上手吗？

答：非常容易理解而且容易上手，且上手后有一种上瘾的感觉。

# HeePHP是什么架构的？

答：Mvc架构的PHP框架。

# HeePHP是否支持自动验证？

答：支持自动验证。

# HeePHP是否支持对数据字段值的自动设置？

答：支持对数据字段值的自动设置，通过模型层定义 set_自动名(初始值) 返回设置的值即可。

# HeePHP是否支持数据缓存？

答：支持数据缓存，通过调用Cache类或cache函数设置或获取缓存。

# HeePHP视图层是原生PHP代码吗？

答：是的，原生代码有很好的执行效率，且良好的代码结果会使得页面整洁而且易读。

# HeePHP框架安全吗？

答：安全，系统框架提供了数据过滤等安全措施，以防止数据库注入，脚本注入，上传文件注入等，用户也可自行附加相关功能。
