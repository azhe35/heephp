    # 验证码

    HeePHP 提供的验证码功能，可以提供验证码展示，验证码的输入效验

    # 验证码生成

    使用randChar生成验证码 参数列表：

    参数名 类型 默认值 说明
    len int 4 位数
    fotmart string all 可选：all 所有 upper 大写字母 lower 小写字母 char 大小写字母 number 数字
    
$code = randChar(6);

$code = randChar(6,'number');

$code= randChar(5,'lower');

    # 验证码展示

    调用函数vercode展示验证码

    参数列表：

    参数名 类型 默认值 说明
    code string 空 验证码
    fontsize int 20 字体大小
    width int 80 验证码宽度
    height int 50 验证码高度
    linecount int 12 干扰线数量

    生成验证码：

    class index extends controller{
    //控制器方法
    public function vcode(){
        //生成5位小写字母
        $code= randChar(5,'lower');
        vercode($code);
    }
}

    使用验证码：

    <html>
  <body>
    <form action="<?=url('action')?>">
      验证码：<input type="text" name="vcode" /><img src=<?=/app/index/vcode?>>
    </form>
  </body>
</html>

    效验验证码

    使用 checkvcode 函数效验验证码输入是否成功，带有一个参数：提交的表单域名，默认为 vcode

    class index extends controller{
    //控制器方法
    public function action(){
        if(checkvcode()){
            //验证成功
        }else
        {
            //验证失败
        }
    }
}
