# 发送邮件

HeePHP 支持邮件发送，邮件的发送可以通过配置文件，也可以将配置传入函数。

配置文件\app\confing.php的mail项发送邮件的配置

//发送邮件配置
'mail'=>[
        //发送邮件服务器
        'server'=>'',
        //登录名
        'username'=>'',
        //登录密码
        'password'=>'',
        //发送人的邮箱
        'form'=>''
    ]

发送邮件的函数为sendmail

参数列表

参数列表

参数名 类型 默认值 说明
to string 空 接收人
subject string 空 邮件标题
body string 空 邮件内容
attachment string 空 附件文件
conf array 空 发送邮件的配置，如果为空则使用配置文件配置，配置项与系统配置相同
