    # 上传

    HeePHP 具有上传文件功能，使用上传函数能轻松的调用上传功能并对上传的文件格式、文件大小进行验证

    # 单文件上传

    调用函数 uploadfile()

    参数列表：

    参数名 类型 默认值 说明
    fname string 空 表单域的名称
    allowedExts array 空 允许上传的文件扩展名字符串数组
    allowfilesize int 空 允许上传的文件大小，单位字节
    dir string 空 上传到的目录
    nametype string md5 新生成文件名的规则：md5,timespan,guid

    返回值：
    return ['name'=>文件名,'ext'=>扩展名,'dir'=>相对路径,'fullpath'=>保存的绝对路径]

    代码参考：

    //当天的文件存入一个目录
$dir = '\upload\\'.date('Ymd');
//上传文件使用guid保存文件名
$info = uploadfile('file1',['jpg','gif','png'],1024*50,$dir,'guid');
//如果发生错误
if(isset($info['error'])){
    $this->error($info['msg']);
}

    # 多文件上传

    调用函数 mulit_uploadfile()

    参数列表：

    参数名 类型 默认值 说明
    fname string 空 表单域的名称
    allowedExts array 空 允许上传的文件扩展名字符串数组
    allowfilesize int 空 允许上传的文件大小，单位字节
    dir string 空 上传到的目录
    nametype string md5 新生成文件名的规则：md5,timespan,guid

    返回值：
    return ['success' => 成功的列表, 'fail' => 失败的列表]; 代码参考：

    //当天的文件存入一个目录
$dir = '\upload\\'.date('Ymd');
//上传文件使用guid保存文件名
$re = mulit_uploadfile('file1',['jpg','gif','png'],1024*50,$dir,'guid');
