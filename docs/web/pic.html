# 图片处理

HeePHP带有强大的图片处理功能：可进行水印、旋转、缩放等操作。

示例：

<?php
try {
  // 创建图片处理对象
  $image = new \heephp\heeimages();

  // 调用 ✨
  $image
    ->fromFile('image.jpg')                     // 载入图片
    ->autoOrient()                              // 根据exif数据调整方向
    ->resize(320, 200)                          // 缩放为320X200的图片
    ->flip('x')                                 // 水平翻转
    ->colorize('DarkBlue')                      // 深蓝色
    ->border('black', 10)                       // 增加10像素的黑色边框
    ->overlay('watermark.png', 'bottom right')  // 在右下角增加水印图片
    ->toFile('new-image.png', 'image/png')      // 将图片保存到new-image.png,并转为png
    ->toScreen();                               // 输出到浏览器

  // 处理错误 💪
} catch(Exception $err) {
  // 获取错误信息
  echo $err->getMessage();
}
