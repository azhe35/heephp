# 图片处理

HeePHP 带有水印功能，可以添加文字、图片水印，且位置可以自行决定。

使用imageWaterMark函数进行图片加水印，该功能依赖 GD 库

生成图片后会将原有图片覆盖，无返回值

参数列表

参数名 类型 默认值 说明
originalImage string 空 原始图片位置(必须)
waterPos int 5 水印位置 1 左上 2 上中 3 上右 4 左中 5 居中 6 中右 7 左下 8 下中 9 右下
waterImage string "" 水印图片位置
waterText string "" 水印文字
textFont int 5 水印文字字体大小
textColor string #ffffff 水印文字颜色
fontFile string arial.ttf 水印文字字体 需将字体放到\public\assets\fonts 目录

如果存在水印图片，则优先使用图片水印

//生成水印图片 文字水印 为HeePHP字体大小为20 居中的图片
imageWaterMark(ROOT.'/public/uploads/abc.jp',5,'','HeePHP',20);
