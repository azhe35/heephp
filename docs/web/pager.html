# 分页

# 调用模型的 page 方法达到分页的效果

分页的信息通过模型的 pager 属性获取

$model->pager

其返回数组，成员如下：

名 说明
count 总记录数
pagesize 每页的记录数量
page 当前页码 从 1 开始
pagecount 总页数
show 系统默认的分页 HTML 代码

获取：

$users=model('users');
$users->select();
$p = $users->pager;

//获取分页信息
$count = $p['count'];
$pagesize=$p['pagesize'];
$page=$p['page'];
$pagecount=$p['pagecount'];
$show=$p['show'];
