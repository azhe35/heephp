# 数据验证

HeePHP 验证规则使用字符串拼接

不进行配置，则不进行数据验证

HeePHP 使用自动数据验证，需要配置 4 个模型变量：

变量名 规则 说明
insert_validata 字段 1|规则 1+规则 2+规则 3+... 插入数据时验证规则
update_validata 字段 1|规则 1+规则 2+规则 3+... 更新数据时验证规则
insert_message_validata 字段 1|规则 1+规则 2+规则 3+... 插入数据时验证规则错误消息
update_message_validata 字段 1|规则 1+规则 2+规则 3+... 更新数据时验证规则错误消息

配置验证规则

    //更新数据时的验证规则：title字段必须输入，context字段必须输入
    protected $update_validata = "title|must;context|must;";
    //更新数据时验证错误的提示消息：title为标题，规则为必填；context为内容，规则为不能为空
    protected $update_message_validata = "标题|必填;内容|不能为空;";


    //插入数据时：username为必填、唯一、字母数字下划线组合且最小为6位
    protected $insert_validata='username|must+unique+alphaNumDash=6;';
    //插入数据时验证失败时的提示消息，对应上一行验证规则
    protected $insert_message_validata='用户名|必填+已存在该用户名+字母数字下划线最少6位;';

    //更新数据时：nickname为必填字母数字下划线组合最少6位最多18位 ，email为必填且是一个email地址
    protected $update_validata="nickname|must+alphaNumDash=6,18;email|must+email;";
    //更新数据验证错误时的提示，对应上一行验证规则
    protected $update_message_validata="昵称|必填+字母数字下划线最少6位最多18位;email|不能为空+必须输入电子邮箱;";

单独验证：

    //从post中获取数据
    $data=request('post.');
    //创建验证并传入要验证的数据
    $validata = new validata($data['password']);
    //判断验证是否通过
    if(!$validata->must()||!$validata->alphaNumDash(6)){
        //设置验证错误信息
        validata::set_error(['message'=>'密码不能为空，请输入6位以上字母数字下划线组合！','field'=>'password','title'=>'验证错误']);
        //显示验证错误消息
        validata::showerror();
        //终止
        exit;
    }

# 验证规则

验证规则 参数 说明
must 空 必填
int [min=int.min,max=int.max] 整数
double [wei=0] 小数，小数位数
alphaNumDashChinese [min=1,max=int.max] 字母数字中文下划线
alphaNumDash [min=1,max=0] 字母数字下划线
alphaNum [min=1,max=0] 字母数字
alpha [min=1,max=0] 字母
mobile 空 手机号
tel 空 电话
postcode 空 邮编
email 空 电子邮箱
qq 空 QQ 号码
equal [value] 相等，传字段（验证规则）或值（独立验证）
notequal [value] 不相等，传字段（验证规则）或值（独立验证）
idcard 空 身份张
url 空 网址http://或https://开头
ip 空 ip 地址
chinese 空 中文
unique 空 数据库字段值唯一
regex [正则] 自定义正则
