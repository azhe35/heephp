# 分页参数

# 当一个页面存在多个需要分页的表格时，需要设置此参数，以区分不同表格的URL分页参数

pageparm()设置分页的URL分页参数名，参数1个：URL分页参数名 默认值：page
需配合page()使用

示例：


$ulist = model('users')->pageparm('p')->page();

$plist = model('product')->pageparm('pag')->page();
