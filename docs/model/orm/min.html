# 查询最小值

min()查询最小值，参数2个字段、字段别名
需要配合value()、get()、group()使用

示例：

//返回所有用户中年龄最小的用户年龄
$minage = model('users')->min('age')->value();

//返回最小价格商品的价格
$minprice = model('product')->min('price')->value();

//返回不同分类的最小价格的商品列表
$list = model('product')->group('category')->min('price')->all();
