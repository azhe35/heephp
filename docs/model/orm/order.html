# 排序

order()对结果进行排序，参数一个排序字段以及顺序\

示例：

//根据年龄倒序查找所有数据
$list=model('users')->order('age desc')->all();

//分页查找按照ID正序排列结果
$list = model('product')->order('id asc')->page();
