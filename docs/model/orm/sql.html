# 输出SQL语句

sql()仅输出SQL语句，不执行SQL语句\

示例：


$sql = model('users')->sql();

$sql = model('users')->where("id=10")->sql();

$sql = model('users')->where('age>20')->whereAnd("city='北京'")->sql();
