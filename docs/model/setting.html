# 配置模型

模型的配置都有默认值，如果没有进行配置，那么值将采用默认值

如果没有手动创建模型文件，那么会自动创建具有默认值的模型。

建议手动创建模型文件，会提升系统性能

模型的配置参数如下：

配置不是必须的，如果未进行配置则使用默认值。

//定义users模型
class users extends heephp\model{

    //是否开启自动时间写入
    protected $autotimespan=false;
    //是否开启软删除
    protected $softdel=false;
    //自动写入创建的字段名
    protected $field_createtime='create_time';
    //自动更新创建的字段名
    protected $field_updatetime='update_time';
    //自动删除创建的字段名
    protected $field_deletetime='delete_time';


    //插入数据时的验证规则
    protected $insert_validata="";
    //修改数据时的验证规则
    protected $update_validata="";
    //插入数据时的验证消息
    protected $insert_message_validata="";
    //修改数据时的验证消息
    protected $update_message_validata="";

    //表前缀
    protected $table_prefix;
    //模型关联表
    protected $table;
    //关联表的主键
    protected $key='';

}

详细配置说明：

变量名 默认值 类型 说明
autotimespan false bool 是否开启自动时间写入
softdel false bool 是否开启软删除
field_createtime create_time string 自动写入创建时间的字段名
field_updatetime update_time string 自动写入更新时间的字段名
field_deletetime delete_time string 自动写入删除时间的字段名
insert_validata 空 string 参见验证
update_validata 空 string 参见验证
insert_message_validata 空 string 参见验证
update_message_validata 空 string 参见验证
table_prefix 空 string 为空时使用 config.php 配置中的表前缀
table 空 string 为空时使用当前模型的名称
key 空 string 为空时自动获取表主键
