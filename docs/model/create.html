# 创建模型

模型是 HeePHP 中重要的概念

自动验证 软删除 自动数据 关联模型 都来自模型

模型的创建与使用简单 易于理解，如果能熟练掌握，系统的开发会变得轻易而举

# 模型文件位置

模型文件不是必须的，如果没有模型文件，那么会自动创建具有默认值的模型

# 单应用

\app\model\{模型名}.php

# 多应用

\app{APP}\model\{模型名}.php

# 模型文件的定义

模型文件名为小写字母开头的字母下划线(_)组合 通常为数据表名.php 例如 users.php menus.php sys_menus.php

//当前模型的命名空间 通常定为：\app\{APP}\model
namespace app\admin\model;
//使用框架的model类
use heephp\model;
//继承框架的model作为基类
class menus extends model
{

}
