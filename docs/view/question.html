# 常见问题

# 为什么要使用原生 php 代码作为模板？是否有其他模板引擎？

答：HeePHP 追求简单、效率。
原生 php 代码本身就是很好的模板引擎，对于效率来说也是最优的。目前没有其他模板引擎。

# 是否可以在模板中使用包含其他文件？

答：建议使用 import 函数。
参数为被包含文件的路径(/开头：相对于 view 根目录路径，否则相对于当前模板文件路径)。

# 模板变量赋值的方法

答：控制器中调用$this->assign('var',$var)。

# 模板运行的速度快吗？

答：因为是 PHP 原生的模板，省去了编译判断等步骤，使得程序运行变的快了不少。

# 是否可以生成静态页面？

答：可以，fetch redrict erro success 等方法都会返回页面的内容，可以自行开发。

# 是否支持 Layout？

答：支持，需要根据用户自己的需求自定义 Layout，需要结合 import 函数。
