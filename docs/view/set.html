# 变量赋值

在控制器中使用如下代码变量赋值

$this->assign('name',$name);
$this->assign('title',$title);
$this->assign('item',['id'=>1,'name'=>'tigger']);

在模板中使用变量

<a href="/" title="<?=title?>"><?=name></a>

<ul>
<?foreach($item $k=>$v){?>
    <li><?=$k?>-<?=v?></li>
<?}?>
</ul>
