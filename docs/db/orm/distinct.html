# 返回字段唯一值

distinct()返回列的唯一值，参数一个字段名
需配合value()、select()、get()、all()使用

示例：

//取商品属性表，商品编号数组
$list = table('product_attr')->distinct('product_id')->value('product_id');

//返回唯一商品编号列表
$list = table('product')->distinct('id')->all();

//返回每一个班级一个班干部
$list = table('student')->distinct('`class`')->where('cadre=1')->all();
