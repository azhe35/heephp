# 取值

value()取值，参数可选1个：字段名
返回：一条记录返回字符串值，多条记录返回字段数组列表

示例：

//返回id等于1的用户的username
$username =table('users')->where('id=1')->value('username');

//获取年龄大于18的用户总数
$count = table('users')->where('age>18')->count('id')->value();
