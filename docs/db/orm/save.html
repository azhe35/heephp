# 保存数据

通过save方法可以保存数据

如果主键为空，则新增数据
如果主键有值，则更新数据
返回主键值或false

示例一：

$data=request('post.');
$id=table('users')->save($data);

if($id){
    return $this->success('保存成功',url('manager'));
}else{
    return $this->error('保存失败');
}

根据条件保存数据：
先根据条件查询
如果存在记录则更新条件匹配的记录
如果不存在条件匹配记录则插入数据

示例二：


$id=table('users')->where("isread=0")->save($data);

if($id){
    return $this->success('保存成功',url('manager'));
}else{
    return $this->error('保存失败');
}
