# 调用切面方法

定义完标记位后，需要定义标记位相关的类，以实现切面：

切面需要调用的方法集合文件位置：\app\aop.php

# aop.php 文件定义

return [
    'app_init'=>'\app\aop\systeminfo',
    'app_init'=>'\app\aop\info',
    'app_end'->'\app\aop\footer'
]

以上代码定义了：不同标记位需要调用的类

# aop 类定义

类目录为：\app\aop
命名空间：app\aop
需要有带一个参数的 run 方法 并返回值

//命名空间需为app\aop
namespace app\aop;
//定义类
class systeminfo {
    //定义run方法 和 引用类型参数
    public function run($parms){

        //处理结束后需要将值返回
        return $value;
    }

}
