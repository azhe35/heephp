# 输出

控制器需要将内容输出到浏览器，为此提供了一些方法

//直接将文本输出到浏览器 路由：\{apps}\{controller}\info\1 输出：info:1
function info($id){
    return 'info:'.$id
}

//渲染模板文件后输出到浏览器 路由：\{apps}\{controller}\info
function info(){
    return $this->fetch();
}

//渲染模板文件\{app}\view\users\info.php后输出到浏览器 路由：\{apps}\{controller}\info
function info(){
    return $this->fetch('/users/info');
}

//渲染模板文件\{app}\view\{controller}\show.php后输出到浏览器 路由：\{apps}\{controller}\info
function info(){
    return $this->fetch('show');
}

//输出Json字符串输出到浏览器 路由：\{apps}\{controller}\info
function info(){
    $arr=['key1'=>'value1','key2'=>'value2'];
    return $this->json($arr);
}


//在浏览器下载文件 路由：\{apps}\{controller}\info
function info(){
    $file='./../public/asset/info.zip';
    $fname = 'info.zip';
    return $this->file($file,$fname);
}
方法 参数 解释
直接放字符串 字符串 直接将字符串返回到浏览器
fetch (空 或 字符串) 渲染模板后输出
json (字符串 或 数组) 输出 Json 字符串到浏览器
file (服务器文件路径，下载后默认文件名) 将服务器文件输出下载到浏览器
