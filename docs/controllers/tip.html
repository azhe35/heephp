# 成功与失败页面

操作通常会有操作成功或操作失败，此时应该用户以提示，提醒其是否操作成功。

在用户操作后会跳转到操作成功或操作失败页面，几秒内会跳转到指定页面

以下为代码示例：

function save(){
    //从post中获取数据
    $data=request('post.');
    //建立users的model对象
    $users=model('users');
    //插入数据并返回id
    $lastid = $users->insert($data);
    //判断是否插入成功
    if($lastid){
        //成功跳转成功页面 并在短暂停留后，返回manager方法
        return $this->success('保存成功！','manager');
    }else{
        //失败跳转到失败页面 并在短暂停留后，返回上次操作页面
        return $this->error('保存失败');
    }


}
方法名 参数 说明
success (成功提示，跳转方法[,停留时间=1]) 跳转到成功页面并在短暂停留后跳转到指定页面
error (失败提示[,停留时间=1]) 跳转到失败页面并短暂停留后返回上一个页面
