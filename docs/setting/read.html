# 读取配置

配置的读取使用\heephp\Config类操作

使用助手函数config()将简化配置的读取操作

以下读取配置的一般用法


//读取配置
$pagesize = \heephp\Config::get('pagesize');

$pagesize = config('pagesize');

//读取二级配置使用点操作符
$dbname = config('db.db_name');

//读取所有二级配置并返回数组 注意不能少点
$db = config('db.');

以上方法均可正确读取配置
