# 系统常量

常量可以直接拿来使用

变量 值
APPS 是否为多应用
APP 为多应用时，当前请求应用的名称
CONTROLLER 当前请求的 Controller 的名称
METHOD 当前请求的方法名称
PARMS 当前请求的参数列表数组
ROOT 系统根目录路径
